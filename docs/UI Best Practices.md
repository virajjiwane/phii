# Modern UI Best Practices

* Use Material 3: Enable useMaterial3: true for the latest design system
* Consistent spacing: Use multiples of 4 or 8 for padding/margins
* Custom fonts: Google Fonts adds polish instantly
* Smooth animations: Use Hero widgets and implicit animations
* Responsive design: Use MediaQuery and LayoutBuilder for different screen sizes
* Dark mode support: Always provide both light and dark themes
